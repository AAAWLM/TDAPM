## SMDH
The source code of the **T**arget **D**istribution **A**lignment via **P**rogressive **M**atching (TDAPM) framework.

## Paper
**TDAPM: Target Distribution Alignment via Progressive Matching for Multi-Source-Free Domain Adaptation**

Qing Tian, Linmin Wang, Xiang Liu, Junyu Shen, Yuhui Zheng, Jun Wan, Zhen Lei

## Dataset
We used four benchmark datasets as following:

-Office-31
-Office-Caltech
-Office-Home
-DomainNet

## Dependencies
**TDAPM** needs the following dependencies:

- python 3.8.5
- PIL == 9.2.0
- torch == 1.13.0
- numpy == 1.23.3
- torchvision == 0.14.0
